# Ecommerce Backend

Backend API for a full‑stack ecommerce application built with:

- Node.js
- Express
- MongoDB (Mongoose)
- JWT Authentication
- Firebase Cloud Messaging (for notifications)

## Features

- User authentication
- Product management
- Cart and orders
- Payment flow (gateway‑ready)
- Order history
- Notifications

## Setup

```bash
npm install
npm run dev
```
